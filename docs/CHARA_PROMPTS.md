## FaceCam 生成ワークフロー

```bash
# 特定の1枚だけ再生成
rm assets/generated/face/mil/eat_blink.png
python3 tools/gen_face_mil.py

# キャラ丸ごと全部作り直し
python3 tools/gen_face_mil.py --force

# 他キャラ
python3 tools/gen_face_muu.py --force
python3 tools/gen_face_yuzuki.py
python3 tools/gen_face_kiriko.py --force

# 生成後 Godot re-import
/Applications/Godot.app/Contents/MacOS/Godot --headless --import --path .
```
