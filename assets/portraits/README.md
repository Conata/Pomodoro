# キャラクター立ち絵（ポートレート）

ストーリー（会話）とステータス画面で表示される「等身の高いキャラ」。

## 本物の立ち絵を入れる方法

このフォルダに以下のファイル名で PNG を置くだけで、自動的に差し替わります
（コード変更不要。`src/ui/portrait.gd` の `draw_into` が override を優先）。

- `mil.png` … ミル（量産型アンドロイド・前衛守護）
- `yuzuki.png` … ユズキ（人間・近接火力）
- `muu.png` … ムゥ（配信者・手数/歌）
- `kiriko.png` … キリコ（オカルトサイエンティスト・後衛重撃）

推奨：縦長のバストアップ（例 720×1040 以上）、背景透過 PNG。
MIDNIGHT VIDEO / Rain98 系の暗がり＋ネオンに馴染む彩度低めが合います。
